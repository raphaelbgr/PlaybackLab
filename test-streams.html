PlaybackLab - Stream Test Page

Load this page with the extension installed. Open DevTools and go to the PlaybackLab tab.

How to test:
  1. Open DevTools (F12) and find the PlaybackLab tab
  2. Enable Auto-Detect in the extension
  3. Click "Load" on the players below — each should appear as a detected stream
  4. Verify: HLS streams show as HLS, DASH as DASH
  5. Verify: Segments are counted under parent streams (not shown as separate streams)
  6. Verify: Manifest parsing shows variants, audio tracks, and codec info
